实验环境

一台Linux主机

实验步骤

安装软件

初始化数据库并设置root用户的密码(phpmyadmin时需要用到该用户登录，也可自己创建用户进行登录，详细相关命令请查看 MariaDB数据库管理系统)

获取PHPmyadmin包并解压和移动路径

修改配置文件以及修改相关文件的权限

开启Apache服务和防火墙放行以及关闭selinux

Windows网页上测试

注：使用yum安装的php可能低于phpmyadmin所要求的的版本，所以需要下载较高版本的php
我这里更新的是php7.2版本(如果安装的是phpmyadmin4.0以上，则一般都需要升级php版本)
需要将之前安装的php5.几版本卸载(yum remove php)
